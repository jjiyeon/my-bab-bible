# Bob Bible (가칭) — 1차 MVP

따뜻하고 친근한 분위기에서 성경을 읽고, 마음에 드는 구절을 저장하며, 공동체(가족/셀/모임)와 함께 읽기 경험을 나누는 모바일/웹 서비스입니다.

> 현재 리포지토리는 **1차 작업(뼈대/화면 구조 중심)** 을 마무리한 상태로, 기능/연동은 단계적으로 확장합니다.

---

## 주요 기능 (MVP 범위)

- **홈(Home)**  
  - 최근 저장한 말씀 카드
  - 친구가 저장한 말씀 카드(랜덤/샘플)
  - 좋아요(토글 UI), 공유(1차: 비활성/안내)

- **성경(Bible)**  
  - 구약/신약 탭
  - 책 → 장 → 절 탐색
  - 구절 저장(1차: UI 또는 더미 동작)

- **내가 저장한 말씀(Saved)**  
  - 저장한 구절 리스트
  - 여러 구절 묶음 표시(+N)
  - 다이얼로그(모달)로 전체 구절 확인, 좌우 이동, 삭제(1차: UI 또는 더미 동작)

- **같이 읽기(Together Reading)**  
  - 모임 리스트(진행중/완료 탭)
  - 모임 카드(기간/범위/진행률)
  - 모임 상세(참여자별 진행률, 내 상태 강조)

- **마이페이지(My Page)**  
  - 프로필 카드
  - 교회 정보 입력(1차: 텍스트)
  - 다이제스트 설정(스위치)
  - 개발자 후원(1차: UI/안내)

- **로그인(Login)**  
  - Google / Apple / 자체 로그인 UI (연동은 2차)

---

## 기술 스택

- Next.js (App Router)
- TypeScript
- (선택) Tailwind CSS / UI Kit (프로젝트 설정에 따름)

> 앱 래핑(React Native WebView / Capacitor), 푸시, 결제(IAP) 등은 2차 단계에서 확장 예정입니다.
