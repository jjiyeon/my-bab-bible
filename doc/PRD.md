# Bob Bible (성경 같이 읽기) - 상세 기능 요구사항 #
## Overview ##
**What are we building?**

Bob Bible(가칭)은 전 연령층이 친근하게 성경을 읽고, 저장하고, 공동체와 함께 읽기 경험을 나눌 수 있는 모바일/웹 서비스입니다. 홈, 성경, 내가 저장한 말씀, 같이 읽기, 마이페이지, 로그인 등 주요 화면을 제공합니다.

**Why are we building it?**

기존 성경 앱의 딱딱하고 정적인 분위기를 벗어나, 따뜻하고 소셜한 경험을 제공하여 성경 읽기를 일상적으로 만들고, 공동체적 동기부여와 재미를 더하기 위함입니다.

**Who is it for?**

아이들부터 노년층까지, 성경을 읽고 싶은 모든 사용자. 특히 셀/공동체/가족 단위의 같이 읽기 경험을 원하는 사용자.
## Use Cases ##
- 홈: 사용자가 최근에 저장한 말씀과 친구의 말씀을 한눈에 보고, 좋아요/공유 등 상호작용

- 성경: 구약/신약, 책, 장, 절을 빠르게 탐색하고, 원하는 구절을 저장

- 내가 저장한 말씀: 저장한 말씀을 리스트로 관리, 여러 구절 저장 시 다이얼로그로 전체 확인

- 같이 읽기: 모임을 만들고, 멤버별 진행률을 시각적으로 확인하며, 공동체와 함께 성경을 읽음

- 마이페이지: 내 프로필, 교회 정보, 다이제스트 설정, 개발자 후원 등 부가 기능 관리

- 로그인: 구글/애플/자체 로그인으로 간편하게 가입 및 로그인
## Requirements ##
**홈 화면**

| 요소 | 설명 |
|---|---|
| 앱 로고/네비게이션 | 상단에 앱 로고와 메뉴 아이콘 노출 |
| 내가 저장한 말씀 | 최근 저장한 말씀 1구절 카드, 없으면 '저장된 말씀이 없습니다.' 안내 |
| 친구가 저장한 말씀 | 친구가 저장한 말씀 1구절 랜덤 카드 |
| 좋아요 버튼 | 각 말씀 카드에 '좋아요' 버튼 |
| 공유 버튼 | 각 말씀 카드에 '공유' 버튼(1차는 비활성화 또는 안내) |
| 하단 탭 네비게이션 | 성경, 내가 저장한 말씀, 같이 읽기, 마이페이지로 이동 |

**성경 화면**

| 요소 | 설명 |
|---|---|
| 구약/신약 탭 | 상단에서 구약/신약 선택 |
| 책 리스트 | 탭 아래에 책 리스트 바로 노출 |
| 장 리스트 | 책 클릭 시 해당 책 아래에 인라인으로 장 리스트 펼침 |
| 구절 리스트 | 장 클릭 시 해당 장의 구절을 쭉 이어서 아래에 표시 |
| 구절 저장 버튼 | 각 구절 옆에 '저장' 버튼(북마크 아이콘 등) |
| 오프라인 지원 안내 | 상단 배너 등으로 안내 |
| 번역기능 안내 | 필요시 안내 배너 노출 |
| 하단 탭 네비게이션 | 홈, 내가 저장한 말씀, 같이 읽기, 마이페이지로 이동 |

**내가 저장한 말씀 화면**

| 요소 | 설명 |
|---|---|
| 말씀 리스트 | 저장한 말씀을 1구절=1row로 최근순 정렬 |
| 여러 구절 저장 | row에 대표 구절 + '+N구절' 표시, 클릭 시 다이얼로그로 전체 구절 표시 |
| 다이얼로그 | 구절 본문/출처, 좌우 이동(스와이프/버튼), 삭제 기능 |
| 삭제 버튼 | 각 row 및 다이얼로그 내 삭제 버튼 |
| 안내 메시지 | 저장된 말씀이 없을 때 '저장한 말씀이 없습니다.' 안내 |
| 하단 탭 네비게이션 | 홈, 성경, 같이 읽기, 마이페이지로 이동 |

**같이 읽기 메인 화면**

| 요소 | 설명 |
|---|---|
| 모임 만들기 버튼 | 상단 또는 하단에 '+ 모임 만들기' 버튼 |
| 모임 리스트 | 내가 참여 중인 모임 카드(진행 중/완료 구분 탭) |
| 모임 카드 | 모임명, 기간, 읽기 범위, 내 진행률, 전체 평균, 도장깨기/진행률 애니메이션 |
| 안내 메시지 | 모임이 없을 때 '참여 중인 모임이 없습니다. 새로운 모임을 만들어보세요!' 안내 |
| 하단 탭 네비게이션 | 홈, 성경, 내가 저장한 말씀, 마이페이지로 이동 |

**같이 읽기 모임 상세 화면**

| 요소 | 설명 |
|---|---|
| 모임 정보 | 상단에 모임명, 기간, 읽기 범위 등 표시 |
| 전체 평균/내 순위 | 전체 평균 진행률, 내 순위 표시 |
| 참여자 리스트 | 유저별 아바타, 이름, 진행률(%) 및 바로 아래 가로 프로그래스 바 |
| 내 진행상황 | 내 row 강조(배경 등) |
| 하단 탭 네비게이션 | 홈, 성경, 내가 저장한 말씀, 마이페이지로 이동 |

**마이페이지/더보기 화면**

| 요소 | 설명 |
|---|---|
| 프로필 카드 | 프로필 이미지, 이름, 짧은 자기소개 |
| 교회 정보 입력 | 직접 입력 방식(1차) |
| 다이제스트 설정 | 주말 다이제스트 발송 스위치 |
| 개발자 후원 | 후원 버튼(광고 포함 가능) |
| 친구초대 | 2차 기능, 버튼 비활성화 또는 안내 |
| 하단 탭 네비게이션 | 홈, 성경, 내가 저장한 말씀, 같이 읽기로 이동 |

**로그인/회원가입 화면**

| 요소 | 설명 |
|---|---|
| 구글 로그인 | 구글 계정으로 로그인/회원가입 |
| 애플 로그인 | 애플 계정으로 로그인/회원가입 |
| 자체 로그인 | 아이디/비밀번호 입력 방식 |
| 비밀번호 찾기 | 비밀번호 재설정 플로우 |
| 에러 메시지 | 로그인 실패, 중복 계정 등 안내

**공통/기타**
- 모든 화면은 큰 글씨, 명확한 버튼, 따뜻하고 친근한 톤 유지
- 하단 탭 네비게이션은 모든 주요 화면에 고정
- 오프라인 지원(성경 본문)
- 브라우저 번역기능 연동(성경 화면)
- 접근성(고령층, 저연령층 모두 사용 가능)
- 개인정보 보호 및 보안(로그인, 데이터 저장)
- 확장성(2차 기능 추가 용이)
